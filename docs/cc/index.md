# Contact Center Overview
## Overview
Cisco Packaged contact center enterprise deployment model of 2000 Agents use a redundant architecture for high availability and solution serviceability. They provide a comprehensive feature set, including:

* Call processing and call control
* Web chat, email, and callback
* Social media monitoring
* Rich VRU and routing scripting
* Interactive voice and video response unit
* Voice and video recording and streaming
* Agent selection and queuing
* Web-based agent and supervisor desktops
* Comprehensive reporting

Cisco Packaged Contact Center Enterprise is a predefined deployment for simpicity in configuration of the solution and maintaining it
